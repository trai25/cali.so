## 🎯 What was new on the site

1. I added sign-in and comments to article pages:

   ![The desktop interface for article comments](/content/newsletters/1/comments.png)

   ![The mobile interface for article comments](/content/newsletters/1/comments-mobile.png)

   I did not want the comments to work like most comment sections at the bottom of a page, where messages often lose their connection to the article's context.

2. I added a [Guestbook](/guestbook) page:

   ![The Cali.so Guestbook interface](/content/newsletters/1/guestbook.png)

   I reused the comments UI and interactions for it. It turned out pretty well, though I was still wondering whether it needed replies...

3. This Newsletter was also a new feature. Writing it in a Markdown file and sending it from the admin area made the workflow very convenient. If you are curious, you can browse the source code at [https://github.com/CaliCastle/cali.so](https://github.com/CaliCastle/cali.so).
